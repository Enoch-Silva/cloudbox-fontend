Projecto Contacts README

Aplicação web que armazena os contactos do utilizador em uma base de dados na nuvem

Backend -------------------------------------------------

* APi RESTful criada em NodeJS 
* CRUD feito com ExpressJS router
* Node FileSystem para gerenciamentos dos arquivos
* Body-parser para pegar as informações vindas do front
* Mongoose para conectar ao Banco
* Banco de dados noSql MongoDB Atlas

FrontEnd ------------------------------------------------

* Site criado em ReactJS em ambiente Vite
* Axios e FetchAPI para pegar dados da API
* React hooks para gerenciamento de estados
* React Router para roteamento de paginas
* React modal biblioteca usada para criar os modals
* React Icons biblioteca usada nos icones do site 
* CSS Media queries para aplicar Responsividade à aplicação
